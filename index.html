Gazetteer

Gazetteer is a dynamic global country profiler that leverages APIs to present rich demographic, geographic, and economic insights. Key features include:

  • Interactive Map: Built with Leaflet for responsive and intuitive navigation.
  • API Integration: Fetches real-time data for the points of interest & data for each modal
  • Modals: Country Info, Demographics, Weather, Currency, Earthquakes
  • Markers: Cities, Airports, Train Stations, Hotels, Museums

Tech stack: JavaScript, PHP, Leaflet, Bootstrap, jQuery

Company Directory

Company Directory is a versatile and efficient web-based system for managing organizational information. The application is built to simplify the management of personnel, departmental data, and office locations with features including:

  • Comprehensive Database Management: Add, edit, and delete records dynamically for employees, departments, and locations using a streamlined interface.
  • Dynamic Modals: Interactive forms for data input and updates, with modals that dynamically populate fields based on user selections.
  • Real-Time Updates: AJAX integration ensures that changes are reflected instantly without page reloads.
  • Advanced Filtering: Dropdowns and search functionality enable users to filter and retrieve data quickly and efficiently.
  • Mobile-Friendly Design: Optimized for seamless use across devices, with responsive layouts and accessibility in mind,
  • Prepared Statements: To avoid SQL injections, using PHP and MySQL in the backend with prepared statements.

Tech stack: JavaScript, MySQL, PHP, Bootstrap, jQuery

DotDev

DotDev is a modern blog-sharing platform inspired by dev.to, built with Next.js 19, React, and Prisma. Users can create posts using Markdown, interact through comments and reactions, and save posts to a reading list.

  • Markdown Support: Write posts using a feature-rich Markdown editor with real-time preview.
  • Authentication: OAuth login with Google/GitHub & optional 2FA.
  • Real-Time Interactions: React to posts with emojis & engage in discussions via comments.
  • Reading List: Save favorite articles for later.
  • Middleware Protection: Secure routing to restrict unauthorized access.
  • Responsive UI: Fully mobile-friendly with TailwindCSS.

Tech Stack: Next.js 19, React, TypeScript, Prisma, PostgreSQL, NextAuth.js, TailwindCSS
